# Segmenting Immunohistochemical Cells  Images
This is the repository for my course completion work. This work consists of the segmentation of images obtained during immunohistochemistry examination, collected by the dentistry laboratory of the Pontifical Catholic University of Minas Gerais. The original dataset consists of 10 images of size 2048x1532 pixels, and can be visualized on the following [link](https://drive.google.com/drive/folders/1GMCqh2PRp1Hvtug64nlTOS_hu4e44eSQ?usp=sharing).
